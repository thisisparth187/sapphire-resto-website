## âš™ï¸ Installation

1. **Clone the repository**

    Fork the website and then clone it on your local machine

```

cd sapphire-website
```

2. **Install dependencies**

```

```

3. **Run the development server**

```

npm start     # For plain Node.js / Express projects
```

4. **Open in browser**:

